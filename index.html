Adding a New Quiz

Each quiz is a standalone HTML file in the quizzes/ folder. To add a new one:

  1. Create a new file in quizzes/your-quiz-name.html
  2. Add a quiz card in index.html linking to it
  3. Update the stats in the hero (quiz count, question count)
  4. Commit and push — GitHub Pages deploys automatically
